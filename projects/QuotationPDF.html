Quotation PDF Generator


Project Overview

GK Lanka International (Pvt) Ltd is a startup but experienced company that installs solar panel systems. They wanted to generate quotation documents for give their customers. They did it manually using word processing software. But then they wanted to generate quotation documents through a web application. In the first phase, they ask for a simple web application so that their office users can insert quotation information and then generate the quotation PDF document. This was an immediate requirement. They provided me their data input interface as an HTML file and I edited the interface to get inputs and created another file for the backend process, which was generating the quotation PDF. (This project is still developing. Therefore, some functionalities may be disabled).

Technical Review in Brief
Both developed files are written in PHP using HTML,CSS and JavaScript. I used the mPDF library for the pdf-generating part. jQuery is used for smooth page updating processes without affecting other contents.
As the first interface, this web application requires that user to select the Solar System Grid Type as On-grid or Off-grid or Hybrid.
The next interface is to enter the customer's details.
As the third interface, it has to fill Solar System Informations such as System Capacity, Panel Capacity. There are some auto-calculating functionalities.
On the next interface, Installation Details should be entered.
As the last interface, it has to enter the Financial Details about the System. Such as Electricity Billing Category, Scheme type of the Billing system and Monthly Average Electricity Usage. According to those details, the Average Monthly Electricity Bill will be calculated automatically. Finally, the user has to enter the estimated System Amount Cost with Discounts (or without) and submit to generate the PDF.

PDF is designed and developed by me in a user-friendly and attractive manner. There are some complex calculations such as the Recovery Plan table and the Degradation-Cumulative table. I am glad to say that all of them are in with correct results and no errors.

Used Technologies and Prerequisites

  • Programming Languages : PHP, JavaScript
  • Markup Languages : HTML, CSS
  • The mPDF PHP Library is used for creating the PDF file.
  • mPDF should be installed in the project directory before running this web application. You can simply check https://mpdf.github.io/ to know how to install the mPDF library.
  • After installing mPDF, the index.php file can be run to get inputs to generate the PDF.
