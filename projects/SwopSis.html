SwopSIS - An ecofriendly fashion swapping platform for women


Project Overview

An ecofriendly fashion swapping platform for women that lets users swap gorgeous pieces from their closet with curated pieces from a shared community wardrobe.
There are three main user entities in this web application. They are,

  • Super Admin
  • Group Admin
  • User

Users can create or join to the groups. Then within the group, users can share there stuffs as swap or loan. Swap means give the items for lifetime. Loan means give the items as borrowing, when the owner need and get back that item in future. Each group has a group admin, which is selected by the super admin. And every group member can share and request for the items.
Also, there are some other features like user rating system according to the items that share in the groups.

This is a system that owned by STEMsisters, and redeveloping by me as a freelancer, under the coordination of Bluechip Technologies Asia.
